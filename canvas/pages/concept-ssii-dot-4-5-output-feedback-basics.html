Topic 4: State Feedback [continued]

This is the final concept for this topic. This concept is slightly unusual as we wanted to provide a brief overview of the next chapter in the book on "Output Feedback", which covers the common situation when you are not able to directly measure all state variables of a system. Instead we build linear "observers" with the same methodology as we build a linear controller, whose input are the system outputs (i.e., the measurements), and whose output estimates the unmeasured state variables. A specific and general form of this approach is known as the Kalman Filter, which is a topic for a more advanced course.
