Module II: Control System Concepts  (Topics 4–6)

This second module will introduce the three main topics critical to understanding how we implement control systems:

  • state feedback,
  • transfer functions, and
  • frequency domain analysis.

Through these topics you will deepen your understanding of how control systems can be used to influence the dynamics of a system and the characteristics you need to consider when doing so.

  Topic 4: State Feedback

This topic will formalise your understanding of using feedback for creating a control system. We can analyse:

  • whether there are sufficient inputs to "reach" (i.e., effect control over) every state variable,
  • how to use feedback to stabilise a system, and
  • how to use feedback to shape the desired dynamics of the controlled system.

The topic concludes with a brief introduction and summary of Chaper 8 "Output Feedback", which considers the situation when not all state variables are able to be measured.

  Topic 4 conclusion

After reviewing the concepts above and completing the activities, you should now be able to:

  • Understand and calculate the reachability / controllability and observability of linear state space systems
  • Implement full state feedback using Matlab using a pole placement method
  • Augment a linear state space system to use integral action to ensure steady state reference tracking
  • Appreciate the role of an observer / estimator in real-world systems
