Course notes

The collated course slides are available here:

  Collated slides: applied-control-systems.pdf

Please note, this document is not intended to contain comprehensive and standalone reference material — that's what the Å&M textbook is for. These slides are short overviews for each concept (covering most of the sections of the book) and are used to record concept videos which introduce and discuss the key points of the course.

Individual concept slide files

As well as the collated slides provided above, which are likely to be most convenient if you wish to annotate the document along with the videos, the individual concept slides are also provided — these can be found individually with each video as you work through each weeks' material and look like this:

  Slides: acs-1-1-1-intro.pdf

Note that there is a minor difference between the collated notes and the individual slide documents. As the slides are intended for recording videos, they sometimes contain progressive revealing of information on subsequent slides. In the collated document, these are collapsed into a single page.

Course material source files

The LaTeX code and associated source files for the course material is provided through the following repository:

  Github: AUMAG/applied-control-systems

You are welcome to browse and suggest changes by creating a Github Issue. 

Providing open access to academic material is a core part of the philosophy I follow known as Reproducible Research. You can read more about this approach in articles and publications with a few examples listed below.
