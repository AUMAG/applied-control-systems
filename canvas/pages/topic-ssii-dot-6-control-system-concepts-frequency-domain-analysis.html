Module II: Control System Concepts  [continued]

This second module will introduce the three main topics critical to understanding how we implement control systems: state feedback, transfer functions, and frequency domain analysis. Through these topics you will deepen your understanding of how control systems can be used to influence the dynamics of a system and the characteristics you need to consider when doing so.

  Topic 6: Frequency Domain Analysis

We now have sufficient background to analyse dynamical systems and control systems in detail. We will carefully

  • delineate the system being controlled from the controller itself via the Loop Transfer Function concept, then
  • discuss how we can use the Loop Transfer Function to predict the stability and sensitivity of a controller — without needing to implement the controller in practice.

It is critical when designing and deploying controllers that they avoid instability!

  Topic 6 conclusion

After reviewing the concepts above and completing the activities, you should now:

  • Recognise the difference between the loop transfer function and the closed loop transfer function and what they are respectively used for
  • Be able to predict the stability of a closed loop system using the loop transfer function by way of the Nyquist Criterion
  • Use the Nyquist plot to find the stability margins of a controlled system

  Module II conclusion

The second module of this course has provided you with a rich foundation of tools for working with control systems. This includes:

  • Familiarity and mathematical background to represent and think about dynamical systems with both time-domain and frequency-domain approaches
  • Modelling of control systems in state space form (with a brief introduction to observers);
  • The complementary approach of using transfer functions, which are particularly well-suited to analysing the effects of disturbances and measurement noise on a system;
  • The twin approaches of the Bode and Nyquist plots to visualise control system transfer functions in multiple contexts;
  • Using the Bode and Nyquist to infer properties of the control system, including performance and stability.
