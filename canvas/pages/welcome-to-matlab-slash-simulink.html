Matlab and Simulink

 

This course makes heavy use of Matlab and Simulink by Mathworks for modelling and simulating dynamical control systems. You will not be needing to do substantial "programming" style tasks, but by the end of this course you should be able to:

  • Use Matlab and/or Simulink to solve linear and nonlinear differential equations
  • Use Matlab's Control System Toolbox to model transfer functions and design control systems, using a variety of visualisation and simulation tools
  • Use Simulink to design and run a control system using real-time hardware 

The use of Simulink's real-time hardware interfacing capabilities is the core reason that we use Matlab as the platform for this course.

Alternatives and further exploration

Other software packages that are available are listed below. You are welcome to explore these as you learn the content of this course, but we do require the use of Matlab and Simulink for the use of the practicals and graded assessments.

Python: python-control

Python is an open source general purpose programming language that is now enormously popular. It contains scientific programming modules such as Numpy and Scipy, visualisation tools such as Matplotlib and Seaborn, and a rich suite of free modules across almost all programming domains. There is a comprehensive control systems module called python-control that provides similar tools to Matlab.

This library also has a Matlab compatibility module, which provides equivalent commands and syntax to Matlab's control systems toolbox. This is loaded via the import command

from control.matlab import *

Julia: ControlSystems.jl

Julia is an open source mathematical programming language that has gained substantial adoption in recent years due to its rich ecosystem of applied mathematical toolboxes, its elegant syntax and architecture, and fast execution speed (claimed to rival C despite being an interpreted language like Matlab).
