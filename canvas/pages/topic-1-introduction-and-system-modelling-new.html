Overview

This module generalises concepts from Engineering Modelling 1, Engineering Mechanics and Dynamics into the study of Dynamical Systems.

The first topic introduces you to

  • control systems concepts and
  • system modelling.

One of the keys to success in control systems is to recognise that it is built upon the foundation of Dynamical Systems.

Once our dynamical system is modelled with a standardised methodology, we can then approach control systems using a suite of sophisticated tools.

By the end of this topic, you should be able to demonstrate an understanding of the key concepts below and have begun to apply them in the workshop and practicals.

State space models

In this concept, the key concept of the state space model is introduced. State space models are the bedrock of how we think about almost all modern control systems. They allow us to consider not just "single input single output" control systems (for example, the position of a traverse given the voltage provided into its motor), but also "multiple input multiple output" systems such as quadcopters, with four motor voltages to control six degrees of freedom.

This course will only scratch the surface of how to analyse and design controllers for MIMO systems, but the state space approach provides a rich framework for several advanced courses you may take in the future.

Modelling methodology

In this concept, we cover some of the methodologies used for modelling dynamical systems across multiple disciplines. Once you have got the hang of how these dynamical systems are constructed, it will quickly become second nature to recognise certain properties of an ODE or transfer function and link them to certain characteristics of a physical system.

  Conclusion

After reviewing the concepts above and completing the activities, you should be able to demonstrate an understanding of:

  • what a control system is and does (conceptually)
  • the terminology of control systems, in particular: system inputs and outputs, state variables, state space systems, linear time invariant systems, and how these terms are linked together via ordinary differential equations to model dynamical systems.

You should also be able to:

  • identify state variables within a differential equation and how these are used in both linear and nonlinear models of dynamical systems
  • extract key parameters from time domain graphs of simple system models: Integrator Time Delay (ITD) model, First Order Time Delay model (FOTD), and Second Order Time Delay model (SOTD) 
  • create ODE models in Matlab and Simulink and simulate them with inputs such as a Step Response
