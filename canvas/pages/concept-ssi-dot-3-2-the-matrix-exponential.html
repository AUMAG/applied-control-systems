Concept 3.2: The Matrix Exponential

In this concept, we show with some mathematical detail how the solution to the Initial Condition Response for a linear time-invariant system is governed by the so-called Matrix Exponential function, which allows us explicitly relate the properties of the A matrix with the characteristics of the system response.
